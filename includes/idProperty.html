ID Property

(Required) Sets the properties of the unique ID property for the ORM object.

Name of the property ("id" is recommended).

Name of the database column that will map to the id property.

Denotes if id column value should be auto-generated by the database, and if so how.

Name of the database sequence.

The struct of key/value pairs (in the format "{property= courseId}") providing necessary details to the generator.

ORM data type.

Optional: use if you need to specify a database-specific data type.

Denotes if ORM will create a setter function for the id property. Should be false if using a generator.

Specify the length of the column.
